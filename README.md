How to build
============

Things that can't be missed in order to compile successfully are listed below:

Windows
-------

- Use Visual Studio 2008 (9.0)
- Unpack the following files located at **toolkit/Source/src/gs2d/vendors**:
  - vendors.zip

Mac OS X
--------

- Use Xcode
- Unpack the following files located at **toolkit/Source/src/gs2d/vendors**:
  - vendors.zip
  - Cg.framework.zip
  - SDL2.framework.zip
